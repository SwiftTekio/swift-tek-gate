# SwiftTek Gate

**About**

A modern React application for SwiftTek Gate, featuring a responsive design with Tailwind CSS and Framer Motion animations.

**Edit the code in your local development environment**

This project is a standalone React + Vite application. You can edit the code locally and see changes in real-time.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `yarn install` or `npm install`

**Run the app:**

```bash
yarn dev
# or
npm run dev
```

**Build for production:**

```bash
yarn build
# or
npm run build
```

**Project Structure**

- `src/components/` - React components
- `src/pages/` - Page components
- `src/assets/` - Local assets (images, fonts, etc.)
- `src/lib/` - Utility functions and contexts
- `src/hooks/` - Custom React hooks
